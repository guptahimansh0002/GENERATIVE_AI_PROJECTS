# Generative AI Projects

A collection of Generative AI projects ranging from foundational concepts to advanced implementations — covering LLMs, RAG, agentic workflows, prompt engineering, and more.

This repository is actively growing. New projects (basic to advanced) will be added over time as I continue learning and building in the GenAI space.

## 🚀 About

This repo serves as a hands-on portfolio of my journey into Generative AI — starting from core concepts like prompting and chains, moving toward RAG pipelines, conversational memory, and agentic systems.

## 🛠️ Tech Stack

- **Languages:** Python
- **Frameworks/Libraries:** LangChain, LangGraph, LangSmith
- **LLM Providers:** Groq, OpenAI, Google Gemini (as applicable)
- **Vector Stores:** Chroma, FAISS
- **Other Tools:** Streamlit, FastAPI (for deployment-ready projects)
